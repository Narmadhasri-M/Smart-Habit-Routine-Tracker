Smart Habit & Routine Tracker

A responsive habit tracking dashboard built using Bootstrap 5.
This application allows users to add habits, set a time with AM/PM,
and view all habits in a clean card-based layout.

Features :

Add new habits, 
Select time + AM/PM, 
Display habits in card layout, 
Responsive UI using Bootstrap 5.

Technologies Used :

HTML5, 
CSS3, 
Bootstrap 5.
